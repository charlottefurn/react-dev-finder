# DEVFINDER

Devfinder is a project inspired by [Frontend Mentor](https://www.frontendmentor.io/challenges/github-user-search-app-Q09YOgaH6)

This project uses the GitHub Users api to fetch and display data.

## Built using  
* HTML / SCSS 
* TypeScript
* React
* Bulma
